### 📘 What Is Equation Hub?

**Equation Hub** is an interactive website designed to help people understand **mathematical functions** and how they work.

Users can:

* Visualize functions as graphs
* Watch educational videos explaining functions
* Explore math concepts in a clear and simple way

The goal is to make math **easier to understand and more visual**.
